Intro

Hi!

My name is Zijing Wang or you can call me Vivian.

I am a creative software engineer with experience in JavaScript, React.js, Redux.js, Express.js, Python, Flask, HTML, CSS, Git, PostgreSQL, Sequelize, SQLAlchemy, AWS, and more. Skilled in creating user-friendly interfaces for web applications and communication platforms with create, read, update, and delete (CRUD) capability. Currently attending the Software Engineering Program (Full-time) at App Academy (the #1 boot camp in the US). Holds a Master's degree in Education from the University of Nevada, Las Vegas.

Project

SmartSplit

SmartSplit is a collaborative project. This project help users to split bill amount, groups and friends. The project is built on Python, Flask, SQAlchemy, React, and Redux.

AirCnC

AirCnC is my first full-stack project, inspired by Airbnb, where users can list houses and add reviews for houses. This project uses an Express and Sequelize backend with a React / Redux frontend.

Catsy

Catsy is an e-commerce web application that focuses on selling and buying cat products all over the world. The project involved Python, Flask, SQAlchemy, React, and Redux.
